# charmlibs.interfaces.example_interface_testing

The `example-interface` testing library. Charms that use `example-interface` should use this library in their state-transition tests.

To install, add `charmlibs-interfaces-example-interface-testing` to your Python dependencies. Then in your Python code, import as:

```py
from charmlibs.interfaces import example_interface_testing
```

See the [library reference documentation](https://documentation.ubuntu.com/charmlibs/reference/charmlibs/interfaces/example_interface) for more.
